# VIM

Use digraphs.

1. Go into Insert Mode.
2. Press `Ctrl + K`
3. and then the two characters from the table below that make the letter.

| Character | Combo |
| --------- | ----- |
| á         | a'    |
| é         | e'    |
| í         | i'    |
| ó         | o'    |
| ú         | u'    |
| Á         | A'    |
| É         | E'    |
| Í         | I'    |
| Ó         | O'    |
| Ú         | U'    |
| ñ         | n~    |
| Ñ         | N~    |
| ü         | u:    |
| Ü         | U:    |
| ¿         | ?I    |
| ¡         | !I    |
