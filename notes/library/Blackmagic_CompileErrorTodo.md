# Finally, rebuild

Confirmed working on Fedora 41 with kernel 6.13.4-200.fc41.x86_64

```bash
sudo dkms build -m blackmagic-io -v 14.4.1a4 -k (uname -r)
sudo dkms install -m blackmagic-io -v 14.4.1a4 -k (uname -r)
```
