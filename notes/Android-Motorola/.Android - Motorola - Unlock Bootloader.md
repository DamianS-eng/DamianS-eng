This document goes over the process of unlocking the bootloader of a Moto Edge Plus 2022.

As a precaution, encrypt the Device ID and the Unique Key before checking to source version control.

# Steps

1. Sign into Motorola Support Page
2. Get Device ID
3. Receive Request Unlock Key
4. Set device in fastboot mode
5. Unlock bootloader using key

## Get Device ID

1. Install the Android SDK and the latest Motorola USB Drivers.
 - Tested so far working on Windows, not sure about Linux, but should be possible...
2. Use command prompt to change directory to the SDK tools, then use
```sh
fastboot oem get_unlock_data
```
3. Receive a character string, split into five lines. Arrange into one continuous string.
4. Supply the Device ID string to the motorola support website. They will verify if the device is unlockable.
5. Retrieve the Request Unlock Key from Motorola Support.

## Unlock Device

1. Turn off device
2. Start device in fastboot mode
> Press and hold power and volume down at the same time, release power and then release volume down.
3. Connect device to computer via USB.
4. Use command prompt to change directory to the SDK tools, then use this to confirm the device is accessible:
```sh
fastboot devices
```
5. Use the following:
```sh
fastboot oem unlock UNIQUE_KEY
```
