# Setup

[EasyEDA Website](https://easyeda.)

## EasyEDA Standard

On the Start screen, under Quick Start, go to New Project.

Create a project name.

# Use

# Design Schematic to PCB

Design -> Convert Schematic to PCB

>Alternatively, use `Alt + P`

The New PCB dialog will pop up. Specify the units, how many layers, the shape outline, the origin coordinates and the distance of the board.

# Order PCBs

## Export Gerber File

Open the file with the generated PCB, then Fabrication -> PCB Fabrication File (Gerber).

> Recommend completing the check first.

The export window is a shortcut to JLCPCP's site, but export the Gerber .zip.
