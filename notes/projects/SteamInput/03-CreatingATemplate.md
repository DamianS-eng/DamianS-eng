# Goal: Show how to create an input template.

And show how templates are useful.

## Steam Controller (2015) - Gordon

### 2D Platformer

> Most "Recommended Layouts" are the generic "Gamepad With Camera Controls" layout, which is only necessary if a Right analog Stick is required.

1. Start with Recommended Template.
1. Change Left Trackpad to Left `Joystick`.
   - If the game does not support a Joystick, maintain it as `Directional Pad`.
   - When using left trackpad for digital movement, turn off "Requires Click".
   - If the game does not use diagonal movement, change the layout to `Crossgate`.
1. Set Left Trackpad's `Click` to `Left Stick Click`.
1. Set Haptic Intensity to `Low` to start. If mastered, set to `Off`.
1. Change Right Trackpad to `Button Pad`.
1. Set buttons to the order as follows:
   - South (V)
   - East (>)
   - West (<)
   - North (^)
1. Rename the bindings starting with the direction and then the action in game.

> Trackpad Haptics are useful to get physically reinforced about the bounds of the trackpad that send output to the game.

> Don't get used to convention (Xbox versus Nintendo layouts). Figure where your fingers naturally rest, and set the most common used commands on that region of the trackpad.
